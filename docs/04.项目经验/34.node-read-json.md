## node如何读取一个json文件内容

> 读取json文件

```js
const path = require('path')
const fs = require('fs')
const filename = path.resolve(__dirname, 'test.json')

const data = JSON.parse(fs.readFileSync(filename))
console.log('data', data) // data { name: 'fangdown' }

```
> 本文案例虽然小， 但是能够解惑长久以来没有注意的疑惑，怎么去读一个文件配置
