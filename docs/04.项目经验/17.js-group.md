## 概况
有一个数组,需要调用java接口进行存储,由于数组长度较大,一次性调用会有问题,需要分组执行

## 实施

将数组按一定数量进行分组,如20个为一组.slice 不会截取不存在的元素


```js
const data = [] // 长数据
const chunkArr = []
for(let i = 0; i< data.length; i+=20){
  chunkArr.push(data.slice(i, i+20))
}
```
