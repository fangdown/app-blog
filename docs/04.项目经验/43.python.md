### 安装（window)
1. 下载地址：https://www.python.org/downloads/
2. 选择安装包进行安装（自定义目录）
3. 添加环境变量（需重启）
4. 输入python -v 可以看到安装成功
###  安装pip（包管理工具）
1. 下载地址https://pypi.org/project/pip
2. 安装python setup.py install
3. 添加环境变量（D:\python36\Scripts）（需重启）
4. 安装包 pip install chardet
