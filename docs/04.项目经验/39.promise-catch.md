## 一道面试题,还需要理解一下的

```js
Promise.resolve(1)
.then((res) => {
  console.log(res)
  return 2
})
.catch((err) => {
  return 3
})
.then((res) => {
  console.log(res)
})
```

## 答案是什么
1,2

> then 和catch都返回的是promise,  为什么没有输出3呢,因为没有错误,没有触发catch
