## 将数据写入到json中
在node开发中， 经常遇到从json中读取数据，把数据写入到json中

那么把这个功能的代码 写出来，没事看看，  高频代码

## 代码
```js
const fs = require('fs');
const path = require('path');

const data = {
  name: 'data'
}
// 将作品信息写入文件
let writerStream = fs.createWriteStream(
  path.join(
    __dirname,
    `../../../douyin-data/user-videos.json`
  )
);
writerStream.write(JSON.stringify(data, undefined, 2), 'UTF8');
writerStream.end();
```
