##  nodejs+typescript项目中添加全局global属性
> 比如增加pp属性


### 无引用其他模块
```js
declare namespace NodeJS {
  interface Global {
    pp: any;
  }
}
```
<strong>需在当前页面进行声明，修改global对象</strong>

### 有引用其他模块
```js
import xxx from 'xxx'
declare global {
  namespace NodeJS {
    interface Global {
      pp: any;
    }
  }
}

```
把这个global.d.ts文件放入到src/types/文件夹下，并修改tsconfig.json
```
{
  "paths": {
    "*": ["node_modules/*", "src/types/*"]
  }
}
```
