# 设计稿验收

每次设计稿验收总是被设计找出若干问题，那么是不是要总结一下呢？

## 自测方向
### UI
- 上下左右是否对齐
- 间距是否正确
- 图标icon（防止类似的误用）
- 字体大小、颜色，一行文字中可能部分文字加粗、颜色特别等
- 圆角检查
- 表格的列头内容对齐
### 交互
- 大屏小屏显示
- 弹窗的关闭和打开

### 20220630验收
- 组件库tooltip间距调整
- icon大小调整
- 图片圆角
- 图片缺失
- 图片不对（系统组件库和提供的svg图片不一致）
- 颜色更改
- 分页显示，超过多少条才显示
- 交互优化
