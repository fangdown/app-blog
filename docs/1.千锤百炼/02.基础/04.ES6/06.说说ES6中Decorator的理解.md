## 是什么
Decorator：装饰器，是一个函数， 用于扩展类方法和类属性
### 特性
- @方法
- 写在类或者属性上面
## 怎么用
### 类装饰
```js
class soldier {}

function strong(target){
  target.Ak = true
}

@strong
class soldier {}
soldier.AK // true

```

### 类型属性装饰
```js

function readonly(target, name, descriptor){
  descriptor.writable = false; // 将可写属性设为false
  return descriptor;
}

class Person {
  @readonly
  name() { return `${this.first} ${this.last}` }
}
```
