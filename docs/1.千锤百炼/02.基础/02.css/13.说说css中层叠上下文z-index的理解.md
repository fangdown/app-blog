## 是什么
层叠上下文：抽象概念， 网页由许多层叠加展示，有前有后，z轴越高，越在前面展示
## z-index用法
1. 同级关系
  - z-index值大的排在前面，
  - z-index相等时，按文档流排列（后面会盖住前面）
2. 父子关系
  - 父元素设置了z-index, 那么子元素永远在父元素上方
3. 同级元素下的子元素
  - 同级元素z-index生效， 其子元素覆盖关系由父元素决定
  - 同级元素的z-index越大，越在前面
