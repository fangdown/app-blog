## 是什么
Chrome浏览器默认中文字体最小为12px
- 英文版没有限制
## 实现方式
- zoom 兼容性差
- tranform:scale 兼容性好
## zoom
- 变焦
```html
<style>
    .p12{
      zoom: 0.8;
      font-size: 12px;
    }
  </style>
  
  <p class="p12">
    zoom:我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本
  </p>
```
### transform
- 要block/inline-block
- 相对中心缩放transform-origin
```html
<style>
    .transform12{
      font-size: 12px;
      transform: scale(0.8);
      transform-origin: left;
      display: inline-block;
    }
  </style>
  <p class="transform12">
    我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本我是文本
  </p>
```
