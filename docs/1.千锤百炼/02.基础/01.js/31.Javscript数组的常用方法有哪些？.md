## 是什么
从以下方面去理解：
- 操作方法
  - 对数组的增删改查
- 转换方法
  - 转成别的数据类型
- 排序方法
- 迭代方法
## 操作方法
### 增
- push 尾加
- unshift 头加
- splice
- concat
```js
let arr = []
arr.push(1) // [1]
arr.push(2) // [1, 2]
arr.unshift(3) // [3, 1, 2]
```
### 删
- pop 尾出
- shift 头出
- splice
- slice
```js
arr.pop() // [3, 1]
arr.shift() // [1]
```
### 改
- splice

### 查
- indexOf
- includes
- find

## 转换方法
- join

## 排序方法
- sort
- revert

## 迭代方法
- map
- forEach
