## 是什么
DOM: 文档对象模型 document object model, 是html和xml的编程接口
> 浏览器页面内容
## 操作
- 创建
  - createElement
  - createTextNode
  - createDocumentFragment
  - createAttribute
- 增加
  - innerHTML
  - appendChild
  - insertBefore
  - setAttribute

- 删除
  - removeChild
- 更新
  - innerHTML
  - innerText
  - textContent
  - style
- 查询
  - querySelector
  - querySelectorAll
  - getElementById
  - getElementsByClassName
  - getElementsByTagName
  - 祖孙关系
    - parentNode
    - childNodes
    - firstChild
    - lastChild
    - nextSibling
    - previousSibling
