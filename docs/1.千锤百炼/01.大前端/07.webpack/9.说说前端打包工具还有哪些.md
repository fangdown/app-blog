模块打包工具
## 有哪些
### Rollup
是一款ES Modules打包器，和webpack类型，但是小巧，```适合打包库```
```js
// user.js
export const name = 'fangdown'

// index.js
import {name} from './user'
export const showName = (name) =>{
  console.log(name)
}
showName(name)
```
执行打包
```
 npx rollup code/fe-engin/Rollup/src/index.js --file code/fe-engin/Rollup/dist/bundle.js
```
- 生成bundle.js
```js
const name = 'fangdown';

const showName = (name) =>{
  console.log(name);
};
showName(name);

export { showName };

```
### Vite
新型前端打包工具
1. 组成
- 开发服务器
- 一套构建指令
2. 特性
- 快速的冷启动
- 即时的模块热更新
- 真正的按需编译
