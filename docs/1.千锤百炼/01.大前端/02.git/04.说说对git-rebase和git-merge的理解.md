## 是什么
- git merge
- git rebase
### 特性
- 都是将一个分支的提交合并到另外一个分支
## git merge
将xxx分支合并到当前分支

```
git merge xxx

```

## 区别
- git merge 会产生新的提交记录， 合并记录按提交顺序交叉

- git rebase 把自己的记录放到最新的记录之后， 提交记录清晰，也容易冲突
