## 是什么
1. HEAD：是一个指针，指向了当前分支， 指向最新提交记录
```
$ cat .git/HEAD
ref: refs/heads/master

$ cat .git/refs/heads/master
7406a10efcc169bbab17827aeda189aa20376f7f
```

2. 工作树： 操作的目录，工作区域

3. 索引： 放置想要提交给git仓库文件的地方， git add添加到索引， git commit 把索引提交到数据库
