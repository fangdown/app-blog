## 是什么
1. fetch： 将远程主机内容拉到本地
2. pull：将远程主机内容拉到本地并合并

```
git pull = git fetch + git merge
```

## 怎么用
## fetch
git fetch <远程主机名> <远程分支名>:<本地分支名>
```
git fetch origin master:temp
git merge temp

```
## pull
git pull <远程主机名> <远程分支名>:<本地分支名>
```
git pull origin master:branchtest
```
## 区别
相同：都是从远程拉取最新代码
不同点： git pull 多了一个merge的行为； fetch更安全一点
