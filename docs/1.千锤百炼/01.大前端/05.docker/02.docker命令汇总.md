## 命令汇总
- docker ps  查看运行中容器
- docker ps -a 查看所有容器
- docker images -查看镜像
- docker rmi images(id) -删除镜像
- docker rm containerid -删除容器
- docker logs containerid -查看容器日志
- docker exec -i -t containerid /bin/bash - 进入容器
- docker stop containerid  -关闭容器
- docker start containerid - 启动容器
- docker build -t name . - 生成镜像名name
- docker run -name xxx -d -p 10000:3000 镜像名称
- docker build -t simple-app:$(git rev-parse --short HEAD) - 根据commit进行维护
- docker build --progress plain --no-cache . - 调试build构建
- docker run -it --rm nginx:alpine sh 进入容器

- docker-compose up --build
