## 是什么
接口interface： 一个接口描述一个对象的属性和方法，但不提供具体创建的方法

```js
interface User{
  name: string;
  age: number;
  getData: () => void
}
```
## 怎么用

- readyonly
- 可选属性
- 任意参数
```js

interface User{
  name: string;
  age?: number;
  readonly data: Object;
  getData: () => void;
  [propName:string]: any;
}
const getUserName = (user: User) => user.name



```

### 继承
可以继承多个接口的属性及方法
```js
interface P {
  name: string
}
interface M {
  good: string;
}
interface C extends P,M{
  color: string
}
const fn = (user: C) => {
  console.log(user.good)
}
```
