## 是什么
枚举类型： 一个对象的所有可能取值的集合
## 数字枚举
```js
enum direction {
  up,    // 0 
  right, // 1
  down,  // 2
  left   // 3
}

enum direction {
  up = 100,     // 100 
  right,        // 101
  down,         // 102
  left          // 103
}
```
## 字符串枚举

```js
enum type {
  subject = 'subject',
  web = 'web'
}

enum week {
  sun = 'sun'
  mon = 'mon'
  tue = 'tue'
  wed = 'wed'
  thu = 'thu'
  fri = 'fri'
  sat = 'sat'
}
```
## 异构枚举
字符串和数字混合，使用较少
