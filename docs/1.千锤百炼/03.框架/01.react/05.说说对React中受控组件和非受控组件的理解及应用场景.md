## 是什么
1. 受控组件
- 受代码控制的组件， 组件的状态全程响应外部数据
- value外部传入，onChange触发回调
2. 非受控组件
- 不受代码控制的组件，组件状态依赖自身
- 不需要value，根据输入自动变化
- 通过ref获取对应的值

## 怎么用
### 应用场景
1. 受控组件
- 表单验证
- 动态输入

2. 非受控组件
- 一次性提交表单
