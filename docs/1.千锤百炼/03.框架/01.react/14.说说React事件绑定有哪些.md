## 是什么
把事件定义绑定到当前上下文中

## 怎么用
- 箭头函数
- constructor
- render中绑定bind
- render中箭头函数

```js
// 箭头函数
const handleClick = () =>{}
```
```js
// constructor（类组件中使用）
constructor(){
  this.handleClick = this.handleClick.bind(this)
}
```

```js
// render bind
render(
  <div onClick={this.handleClick.bind(this)}>
)
```
```js
// render 箭头函数
render(
  <div onClick={e =>this.handleClick(e)}>
)
```
