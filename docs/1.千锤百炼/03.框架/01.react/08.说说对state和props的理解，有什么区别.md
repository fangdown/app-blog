## 是什么
都是数据存储的js对象
## 怎么用
### state
- 在constructor中初始化
- 通过setState进行更新
### props
- 外部传入组件内的数据
- 类型： 字符串，数字，对象，回调函数等

### 区别
- state是组件内数据，props是外部数据
- state是内部管理， props通过回调函数变更
- state可以修改， props不可以
- 都能触发更新
