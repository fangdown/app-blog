## 是什么
> 前端路由：页面url发生改变，页面不刷新更新内容
### 模式
1. history模式
2. hash模式
## 怎么用
### hooks中
useHistory
useParams
useLocation

## 原理
1. history
```js
window.addEventListener('popstate', callback)
```

2. hash

```js
history.pushState()
window.addEventListener('hashchange',callback)
```
