## 是什么
调用setState更正state值
## 怎么用
1. 同步更新
  - settimeout等原生异步事件
  - 原生dom事件
2. 异步更新
  - 事件发生于react合成事件中
3. 批量更新
  - 调用更新的时候会根据isBatchingUpdates变量值
  - true： 把更新值加入到队列
  - false： 立即执行更新
