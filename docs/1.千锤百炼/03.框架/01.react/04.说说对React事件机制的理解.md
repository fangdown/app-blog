## 是什么
React 模拟原生DOM事件所有能力的一个自定义事件对象
### 优势
- 兼容所有浏览器
- 与原生浏览器事件相同接口
- 获取原生方式： e.nativeEvent

## 怎么用
- React注册事件都绑定在document上
- React自身实现了一套事件冒泡机制， 所以e.stopPropagation无效
- React通过队列的形式，从触发的组件向父组件回溯，然后调用jsx中定义的callback
- React有一套自己的合成事件
