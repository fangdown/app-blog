## 是什么
React整个组件生命周期包括从创建、初始化数据、编译模板、挂载Dom→渲染、更新→渲染、卸载等一系列过程

## 怎么用
### 3个阶段
1. 创建
  - constructor
  - getDerivedStateFromProps
  - render
  - componentDidMount
2. 更新
  getDerivedStateFromProps
  shouldComponentUpdate
  render
  getSnapshotBeforeUpdate
  componentDidUpdate
3. 卸载
  componentWillUnmount
