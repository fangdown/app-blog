## 是什么
1. React.memo
和pureComponent一样，为函数式组件服务， 对props进行浅比较， 如果一样就不执行
2. useMemo
在组件内， 对结果值进行依赖缓存
3. useCallback
对组件函数进行依赖缓存


## 怎么用
```js
// React.memo
const app = React.memo((props)=>{
  return <div>111</div>
})
// useMemo
const result = useMemo(()=>{
  return count+1
}, [count])

// useCallback
const onChange = useCallback(()=>{
  // dosomething
}, [count])
// ....
return (
  <Cmp onChange={onChange} />
)
```
