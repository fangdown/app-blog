## 是什么
1. 函数组件： 是一个函数，返回的React节点
2. 类组件： 是一个class，继承了React.Component，返回渲染节点
## 怎么用
### 区别
1. 编写格式不同:函数组件： ```return```,类组件 ```render```
2. 状态管理：函数组件```useState```, 类组件```setState```
3. 生命周期：函数组件```useEffect```,类组件```componentDidMount等```
4. 调用方式： 函数组件直接调用， 类组件new实例
5. this： 函数组件不存在， 类组件有this
