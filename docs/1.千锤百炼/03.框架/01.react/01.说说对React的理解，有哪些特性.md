## 是什么
用于构建用户界面的js库， 提供了UI解决方案
## 怎么用
### 特性
1. jsx语法
2. 单向数据流
3. 虚拟dom
4. 声明式编程
5. 组件component

#### 组件优势
1. 可组合
2. 可复用
3. 可维护
4. 函数式组件、class组件

### 优势
1. 组件开发，提高代码复用率
2. 单向数据流比双向绑定更安全，速度更快
