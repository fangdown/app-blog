## 是什么
### tcp
传输控制协议， 是一个面向字节流的通信协议
### udp
用户数据包协议， 是一个面向数据包的通信协议


## 区别
|       |  TCP   | UDP  |
| ----  |  ----  | ----  |
| 可靠性  | 可靠  | 不可靠 |
| 连接性  | 面向连接  | 无连接 |
| 报文  | 面向字节流  | 面向报文 |
| 效率  | 传输效率低  | 传输效率高 |
| 双共性  | 全双工  | 一对一、一对多、多对一、多对多 |
| 流量控制  | 滑动窗口  | 无 |
| 拥塞控制  | 慢开始、拥塞避免、快重传、快恢复  | 无 |
| 传输效率  | 慢  | 快 |

> 简而言之： tcp理性 udp任性

## 应用场景
tcp:  网页、邮件等
udp: 会议、直播、游戏等
