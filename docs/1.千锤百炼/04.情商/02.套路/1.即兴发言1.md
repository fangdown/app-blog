# 即兴发言

## 突然被邀请发言
- 假如婚礼中突然被主持人邀请说几句话，怎么说？
- 假如在饭局中突然被邀请说几句话，怎么说？


> 下面分享一下面对突然的情况，如何有套路的发言，做到心中不虚

## 套路规律
### 场景
参加新人婚礼，在婚宴中获得了一等奖，司仪活跃气氛，随机找人给新人祝福，我幸运的被挑中了。

### 称呼 
任何场合下，可以先做个自我介绍，和对他人的称呼
```
尊敬的各位领导、各位来宾、各位朋友们，大家中午好，我是谁
```
### 套路：感谢+回顾+祝福（赶回猪）
- 感谢
  - 感谢主持人
  - 感谢主人（这对幸福的新人）
- 回顾
```
  - 记得我和新郎第一次见面就在我的办公室，我们聊了一个下午，当时我们在聊天时，我就感觉这个青年不简单，有想法，有能力，敢作为，同时我们也聊到了婚姻，但是接下来呢，我给大家爆个猛料，是新郎当时的择偶标准，你们想知道吗？
  - 起哄
  - 掌声在哪里
  - 新郎说我未来的妻子要沉鱼落雁、闭月羞花，上得厅堂下得厨房，还要温柔善良，说实话，我当时听完后心里就觉得这个青年各方面都很好，就是在感情上天天真了，哪有这么完美的对象，即使有，能遇到并走在一起的概率简直为0，但是呢，当我今天看到新娘的时候，我知道新郎的理想对象终于找到了。

  我呢，因此也有一个小感慨，希望我们在座的各位可以告诉一下身边的单身朋友一句话。
  
  这句话就是：只要心中有目标，充满爱，坚持，努力不懈怠，即使茫茫人海必定可以寻得真爱。

```
- 祝福
```
祝这对新人新婚快乐、百年好合、早生贵子，同时祝在座的各位来宾及家人能够身体健康、工作顺利、心想事成、万事如意。
```
### 套路：赞美+回顾+祝福（赞回猪）
- 赞美
  - 直接赞美
  ```
  我们面前这位英俊帅气的新郎，绝对是青年人中的佼佼者，年纪轻轻，身价已经过千万了，我们面前的这位漂亮的新娘是万里挑一的大美女，两个人可以说是郎才女貌，非常的般配。
  ```
  - 第三者赞美
- 回顾
- 祝福


### 套路：故事+回顾+祝福（雇回猪）
- 故事
  - 经典故事
  - 故事感悟
```
  张场画眉：西汉时期，身为堂堂首都最高行政长官的张敞由于很爱自己的妻子，就经常在家里给妻子画眉，并且传在京城里无人不知，我曾经看到这个典故时，就在典故的旁边写了八个字感悟，哪八个字呢？眉目传情，眉目传爱。刚才我一直观察这对新人，从他们各自看对方的眼神中我能看出他们彼此都很幸福，我相信新郎新娘婚后肯定会更加的甜蜜和恩爱。
```
- 回顾
- 祝福
### 套路：前人说过的话+回顾+祝福（画回猪）
- 前一位人（或者很多人）说过的话
  - 从他们说过的话中挑一句有感悟的话、有意义的话
```
就在刚才啊， 一位朋友说这对新人能走到一起是缘分，我非常的赞同，茫茫人海这么多人。两个人从相识相知、相恋、再到相爱，不是缘分又是什么呢？但是我再给大家报个料，让大家知道，除了缘分之外还需要做个事情，你们知道吗？
```
- 回顾
- 祝福
### 套路：问2个问题+回顾+祝福（问回猪）
- 问2个问题
  - 和主题相关
```
我想问一下在座的各位，你们想不想知道，为什么新郎能够娶到这么漂亮的新娘？

想知道（怎么没有掌声呢）

```
- 回顾
- 祝福
