## 计算颜色及存储

```js
1Byte = 8bit // 字节-比特
1k = 1024Byte // k - 字节
1M = 1024k // 兆- k
1G = 1024M // G-兆
```

###RGB存储

hex转换rgb
hex： #00ffee ===> 0,255,238

1. R、G、B范围值分别为0-255
2. 每一个信道分别是2的8次方，为一个字节，所以每个显示像素共3个字节，
3. 计算总共有多少个像素乘以3字节则为占用空间大小

### png
1. png8  则表示每个像素有1个字节
2. png24 则表示每个像素有3个字节
3. png32 则表示每个像素有4个字节 有4个通道，最后一个是透明度
