说说对选择排序的理解

# 是什么

选择排序： 分最大或最小原则去排序，

  • 从未排序中选择最小的数放入已排序起始
  • 从未排序中选择最小的数放入到已排序末尾

# 怎么用

function select(arr) {
  const len = arr.length;
  let minIndex, temp;
  for (let i = 0; i < len - 1; i++) {
    minIndex = i;
    for (let j = i + 1; j < len - 1; j++) {
      if (arr[j] < arr[minIndex]) {
        minIndex = j;
      }
    }
    temp = arr[i];
    arr[i] = arr[minIndex];
    arr[minIndex] = temp;
  }
  return arr
}
const arr = [1, 19, 2, 5, 99, 23];
const res = select(arr);
console.log(res); // [ 1, 2, 5, 19, 99, 23 ]

# 场景

时间复杂度高， 不推荐使用
