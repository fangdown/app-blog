## flex布局
> 意为”弹性布局”，用来为盒状模型提供最大的灵活性
```css
.box{
  display: -webkit-flex; /* Safari */
  display: flex;
}
```

### 容器的六个属性
1. flex-direction
2. flex-wrap
3. flex-flow
4. justify-content
5. align-items
6. align-content
   
### 项目的六个属性
1. flex-shrink
2. flex-grow
3. flex
4. order
5. flex-basis
6. align-self
